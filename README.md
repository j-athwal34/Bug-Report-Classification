# Bug-Report-Classification
Using machine learning to classify bug reports into performance/non-performance related bugs

Run using either "python3 br_classification.py" or "python3 svm_classification.py"
For statistical test, run "python3 statistic_analysis.py"

Dependencies:
Install the following
- pandas==2.2.2
- numpy==1.26.4
- scikit-learn==1.4.2
- nltk==3.8.1
- scipy==1.9

For nltk, run "python -c "import nltk; nltk.download('stopwords')" in the console.
